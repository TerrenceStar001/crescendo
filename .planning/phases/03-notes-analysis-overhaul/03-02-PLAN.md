    ```js

    import { validateQuestions } from './questionValidator';
    ```

    **Exports (2 named functions, no default):**

    1. **`buildDrillPrompt(passagePreview, weakTypes, part, mistakesContext)`:**
       - Takes: `passagePreview` (string, plain text), `weakTypes` (array of slugs like `['inference', 'mcq']`), `part` (A/B1/B2), `mistakesContext` (array of `{ qNum, type, userAnswer, correctAnswer }`)
       - Builds a typeDist limited to the students' weak types (equal distribution among them, total 100%)
       - Uses `composeFullPrompt(passagePreview, 3, part, typeDist)` for the base prompt
       - Strips the leading "You are a DSE examiner" prefix and TASK: header from composeFullPrompt output (use regex `.replace(/^You are a DSE.*?TASK:.*?PASSAGE:/ms, '')`)
       - Prepends a custom drill-focused header:

       ```
       You are a DSE English tutor creating TARGETED PRACTICE questions.

       The student struggled with these question types: {weakTypes}.
       Their actual mistakes were:
       {mistakeSummary lines}

       Create exactly 3 drill questions targeting their specific weaknesses.
       Make each question directly address the skill they found difficult.
       CRITICAL: Each question MUST be answerable from the passage alone.
       Focus on the SKILL the student got wrong, not just the question type.
       ```

       - Returns the complete prompt string

    2. **`generateDrills(passagePreview, weakTypes, part, mistakesContext, callAI)`:**
       - Calls `buildDrillPrompt(...)` to get the prompt text
       - Calls AI: `callAI(prompt, { system: systemMsg, temperature: 0.7, maxTokens: 2000, timeout: 30000 })`
       - System message: `'You are a DSE English tutor. Generate exactly 3 practice questions as a JSON array. Each question follows the format: { stem, type, skillTested, paragraphRef, marks, options, correctAnswer, explanation }. Return ONLY the JSON array, no other text.'`
       - Parses response: try `JSON.parse(raw)` — if raw is wrapped in markdown code fences, strip them first:
         ```js
         function parseJSONResponse(text) {
           const jsonMatch = text.match(/\[[\s\S]*\]/);
           if (jsonMatch) return JSON.parse(jsonMatch[0]);
           return JSON.parse(text);
         }
         ```
       - Validates with `validateQuestions(parsed)`
       - Returns `parsed.slice(0, 3)` if valid, `null` on failure
       - Error handling: wrap in try/catch, return null on any error. No console logging.

    ```js
    // idle | generating | ready | answering | answered | failed

    const [drillPhase, setDrillPhase] = useState('answering');
    // Only used when phase === 'ready' to track per-drill answering
    ```

    **Handlers:**

    `handleGenerate` — async, uses existing AI call pattern (ReadingModule.jsx lines 219-234 from PATTERNS.md):
    ```js
    const handleGenerate = useCallback(async () => {
      setPhase('generating');
      try {
        const generated = await generateDrills(
          passagePreview, weakTypes, part, mistakesContext, callAI
        );
        if (generated?.length) {
          setDrills(generated);
          setPhase('ready');
        } else {
          setPhase('failed');
        }
      } catch {
        setPhase('failed');
      }
    }, [passagePreview, weakTypes, part, mistakesContext, callAI]);
    ```

    `handleDrillAnswer(questionId, value)` — sets drillAnswers state
    `handleCheckAnswer(qId)` — toggles showing result for a single drill question

    ```jsx
      <button className="drill-generator__cta" onClick={handleGenerate}>
        Generate Targeted Practice
      </button>
    </div>
    ```

    **generating state:**

    ```jsx
    <div className="drill-generator__loading" style={{ textAlign: 'center', color: 'var(--color-text-muted)' }}>
    ```

    ```jsx

      {drills.map((q, i) => (
        <div key={q.id || i} className="drill-generator__question">
          <QuestionRenderer
            question={q}
            number={i + 1}
            value={drillAnswers[q.id] || null}
            onSelect={handleDrillAnswer}
            showResult={drillPhase === 'answered'}

          />
        </div>
      ))}
      <div className="drill-generator__actions">

            Check All Answers
          </button>
        )}
        {drillPhase === 'answered' && (
          <button className="reading__q-btn"
            onClick={() => { setPhase('idle'); setDrills([]); setDrillAnswers({}); setDrillPhase('answering'); }}>
            Try Again
          </button>

      </div>
    </>
    ```

    ```jsx
      <button className="drill-generator__cta" onClick={handleGenerate}>
        Try Again
      </button>
    ```
