# Plan 12-01: Forgetting Curve Foundation

Create the pure-math half-life forgetting curve module and extend DSE_KEYS.

## Tasks

### Task 1: Create forgettingCurve.js utility
Pure JS module with:
- `forgettingCurve(elapsedDays, halfLife)` → returns recall probability (0-1) using exponential decay: `2^(-elapsedDays / halfLife)`
- `calcHalfLife(prevHalfLife, correct, difficulty)` → returns updated half-life
  - Correct: multiply by 1.3 (capped at 180 days)
  - Incorrect: multiply by 0.5 (floored at 0.5 days)
  - Difficulty adjusts: `base * (1 + difficulty * 0.2)` (difficulty 1-5)
- `scheduleNextReview(halfLife, threshold = 0.5)` → returns recommended days until next review
  - Solve for t where `2^(-t/halfLife) = threshold` → `t = -halfLife * log2(threshold)`
  - For threshold=0.5: `t = halfLife`
- `DEFAULT_HALF_LIFE = 30`
- `HALF_LIFE_RANGE = { min: 0.5, max: 180 }`
- Named exports, pure JS, no React deps
