# Phase 2 Research: Question Quality & HKDSE Format

**Date:** 2026-06-24
**Sources:** chenglish.hk (Reading Skills Guide & Paper 1 Analysis), afterschool.com.hk (DSE Paper 1 guide), HKEAA Assessment Framework 2024, HKEP sample paper marking scheme

## Question Type Taxonomy

### Confirmed HKDSE Paper 1 Question Types (by difficulty)

**Level 2–4 (Foundation):**
1. **Specific Factual Information** — Give/Name/When/Who/Where/What
2. **Numbers and Figures** — Complete table with missing percentages/quantities
3. **Correct/Incorrect Statements** — Tick correct items (variant of TF without NG)
4. **Visual Questions** — MCQ with pictures/diagrams
5. **Sentence/Chart Completion** — Flow chart / table fill (1-3 blanks)
6. **Referencing Questions** — "What does X refer to" (pronouns, reference words)
7. **Signposting Words/Phrases** — Find linking/discourse words
8. **Synonym/Antonym** — Find similar/opposite meaning words
   - MCQ synonym in context
   - Phrase/idiom identification
   - Explain word/phrase in context
   - Suggest a replacement word
9. **True/False/Not Given** — TFNG statements

**Level 4–5 (Intermediate):**
10. **Inferencing Questions** — Imply/suggest/why/infer/explain meaning
11. **Comparison** — Similarities and differences between entities
12. **Views and Attitudes** — Writer's attitude/tone/opinion

**Level 5 (Advanced):**
13. **Main Idea Questions** — Main idea of paragraph, text type, purpose of writing, match subheadings, best alternative title, match quotes to speakers, summary cloze

**Level 5\*\* (Highest):**
14. **Figurative Language** — Irony, metaphor, dilemma/dichotomy

### Additional types from AfterSchool guide:
- Summary Cloze (撮要填充題) — fill blanks with ONE word from passage
- Sequencing/Order of Events
- Proofreading (find mistakes in summary)
- Match speakers to statements
- Short Answer (SQ) / Long Answer (LQ)
- Matching (pair items from two columns)
- Table completion on views/opinions

## Mark Allocation Patterns

### Key finding: Most sub-questions carry 1 mark each
From sample paper marking scheme analysis:
- A "Give TWO examples" question = 2 marks (2 sub-questions, 1 mark each)
- A "Complete the table with missing percentages" = 4 marks (4 items, 1 mark each)
- Summary cloze with 5 blanks = 5 marks (5 items, 1 mark each)
- MCQ = typically 1 mark each
- Reference questions = 1 mark each
- TFNG = 1 mark per statement
- Longer explanation/opinion questions = 2-3 marks
- Proofreading/correction questions = 1 mark per correction

### Mark totals vary by year
- Sample paper (PP-DSE): Part A = 51 marks (Q1-19), B1 = 46 marks (Q20-35)
- Current consensus: 42 marks per part
- Question count range: 19-27 questions per part (more sub-items in earlier years)

### Mark distribution pattern
- Part A: mix of short-answer factual (60%), MCQ (20%), TFNG/Tick (10%), reference (10%)
- Part B1: more scaffolded, simpler questions, cloze/matching heavy
- Part B2: more inferencing/opinion, summary cloze, longer answers

## How DSE Questions Map to Passage

- Questions follow passage order (first question covers early paragraphs, last covers end)
- Some questions specify line/paragraph numbers
- Summary cloze draws from a defined range of paragraphs
- TFNG statements typically reference a specified paragraph

## Distractor Patterns (from exam tips)

### MCQ distractors use:
- **Synonyms as wrong answers** — exact keywords from text placed in wrong options
- **Literal meaning traps** — for idiom/phrase questions, the literal meaning is a wrong option
- **Overgeneralization** — "always/never/only/most" often signal False in TFNG
- **Out-of-scope** — option is factually true but not in passage
- **Numerical confusion** — similar numbers/dates/statistics

### TFNG patterns:
- True: indicators are synonymous with text
- False: indicators contradict the text
- Not Given: indicators don't 100% match, or statement overgeneralises
- Overgeneralising words that often signal False: always, never, only, most, a few, more/less, earlier/later

## DSE Marking Scheme Conventions (from marking scheme samples)

- Answer key uses: T/F/X (where X = NG/Not Given in some schemes)
- Acceptable alternatives listed (e.g., "hot water // heated water")
- Spelling must be exact — minor errors lose marks
- Some questions need candidates to "use their own words" — direct copy not accepted
- Part of speech must match (noun→noun, verb→verb)
- For reference questions: "Do not copy a whole sentence"
- For summary cloze: "3C strategy" — Copy, Change (part of speech), Create (contextual inference)

## Recommendations for Implementation

### Question type distribution per part (research-backed estimate):
- Short answer / factual info: 25-35%
- Cloze / gap-fill: 15-25%
- MCQ: 10-20%
- Matching: 8-15%
- TFNG: 8-12%
- Reference questions: 5-10%
- Table/chart completion: 5-10%
- Inferencing / opinion: 5-10% (B2 only)

### Mark-per-question rule:
- Most items: 1 mark
- Multi-item questions: marks = number of correct items
- Extended response (Level 5/B2 only): 2-3 marks
- Total = exactly 42 marks per part

### Question difficulty progression:
- Part A: 70% Level 2-4, 20% Level 4-5, 10% Level 5
- Part B1: 80% Level 2-4, 20% Level 4-5
- Part B2: 20% Level 4-5, 40% Level 5, 40% Level 5\*\*
