<objective>
Integrate IELTS band display into the correction UI, add conversion table settings, and add CSS styling — making the IELTS-first grading pipeline visible and configurable for users.

**Purpose:** Students need to see their IELTS band as the primary score (since that's what the AI grades accurately) alongside the DSE equivalent. Power users and teachers need to adjust the conversion mapping if they disagree with the calibration. The validation tool allows spot-checking the conversion against real candidate scripts.

**Output:**
- WritingModule.jsx: IELTS band badge in correction summary (Part A, Part B, and combined views)
- SettingsPage.jsx: IELTS→DSE conversion table configuration (collapsible section in DSE tab)
- App.css: IELTS badge styling, conversion table settings section
</objective>
