<objective>
Rewrite the AI correction pipeline from direct-HKEAA scoring to IELTS-first scoring with automatic conversion back to HKEAA.

**Purpose:** LLMs produce more accurate scores in IELTS Writing band format (0-9 per TA/CC/LR/GRA) than in direct HKEAA Content/Organisation/Language format (0-7). The correction pipeline now uses IELTS as the primary scoring schema internally, then converts to HKEAA scores for display.

**Output:**
- `src/utils/ieltsToDseMap.js` — calibration mapping table with localStorage persistence
- `src/utils/hkeaaWritingRubrics.js` — structured HKEAA level descriptor constants
- `src/hooks/useDSEPapers.js` — buildCorrectionPrompt rewritten with IELTS descriptors; parseCorrectionResponse extracts IELTS bands + converts; combineCorrections works with converted scores
- `src/utils/dseGrading.js` — re-exports conversion utilities from ieltsToDseMap.js
</objective>
