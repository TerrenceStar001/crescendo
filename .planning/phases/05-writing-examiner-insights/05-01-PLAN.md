<objective>
Redesign the AI correction prompt to grade in IELTS bands first, then derive HKEAA Content/Org/Lang scores from the IELTS assessment — fixing grade inflation by using rubrics LLMs handle accurately.

**Purpose:** LLMs do not understand HKDSE standards deeply enough to grade authentically. IELTS Writing band descriptors are globally recognized with abundant LLM training data. By prompting for IELTS bands first, the AI produces more accurate assessments. The HKEAA-equivalent scores (Content/Org/Lang) are then derived from the IELTS assessment, maintaining backward compatibility with the existing correction UI.

**Output:**
- Updated `buildCorrectionPrompt()` with IELTS Writing Task 2 band descriptors as primary rubrics
- Updated `parseCorrectionResponse()` handling new `ielts` block with per-criterion bands 0-9
- Updated system message and correction pipeline wiring in WritingModule.jsx
</objective>
