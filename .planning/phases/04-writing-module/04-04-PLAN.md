## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTML essay content → annotation renderer | Essay text with inline spans for annotations — no innerHTML for user content |
| AI annotation data → tooltip render | AI-generated correction/explanation strings rendered as text |
| IndexedDB history load | Past session data loaded from client-side storage |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-12 | Spoofing | Annotation rendering | mitigate | Annotations wrap matched text in `<span>` with React JSX (auto-escaped). Never use dangerouslySetInnerHTML for user essay text. Use element.textContent for tooltip content. |
| T-04-13 | Tampering | IndexedDB session history | accept | Client-side only; no server-side processing of session data |
| T-04-14 | Denial of Service | History load with large dataset | mitigate | Limit history to 50 sessions (enforced in Plan 03 saveSessionToHistory). Lazy-load only session metadata for list view. |
| T-04-15 | Denial of Service | Annotation matching performance | mitigate | Limit annotations to 50 per session; use string replacement with simple text matching, not regex on large strings |
</threat_model>
