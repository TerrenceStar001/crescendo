Create `src/hooks/useAssessment.js` with default export function `useAssessment()`:

**Data model:**

Assessment profile shape stored at DSE_KEYS.ASSESSMENT:
```js
{
  _initialized: true,
  completedAt: 'ISO string or null',
  selfRatings: {
    reading: { level: 1-5|'not-rated', confidence: 'low'|'medium'|'high' },
    writing: { level: 1-5|'not-rated', confidence: 'low'|'medium'|'high' },
    listening: { level: 1-5|'not-rated', confidence: 'low'|'medium'|'high' },
    speaking: { level: 1-5|'not-rated', confidence: 'low'|'medium'|'high' },
  },
  quizResults: {
    reading: { score: 0-3, questions: [], completedAt: null },
    writing: { score: 0-3, questions: [], completedAt: null },
    listening: { score: 0-3, questions: [], completedAt: null },
    speaking: { score: 0-3, questions: [], completedAt: null },
  },
  finalLevels: {
    reading: { selfRated: 1-5, quizAdjusted: 1-5, final: 1-5, method: 'self'|'quiz'|'adjusted' },
    // ... same for writing, listening, speaking
  },
  _overallDse: '1'|'2'|'3'|'4'|'5'|'5*'|'5**',
}
```

**State:** profile (loaded from IndexedDB on mount), isLoaded flag, activeSkill ('reading'|'writing'|'listening'|'speaking'|null), currentPhase ('self-rate'|'quiz'|'results'|'complete').

**Verification quiz questions (ASMT-02):** Hardcode 3 questions per skill (12 total) as a static constant inside the hook file. These are curated, not AI-generated — they must work offline per the offline-first constraint. Each question:
```js
{ id: number, skill: string, stem: string, type: 'mcq', options: [{label, text}], correctAnswer: string (letter) }
```

3 questions per skill. They should be representative DSE-style questions that test basic competency in each skill area. For example:
- Reading: test comprehension of main idea, detail, inference
- Writing: test knowledge of text type conventions, register, structure
- Listening: test understanding of spoken content, note-taking
- Speaking: test pronunciation awareness, presentation structure, interaction strategies

**Key functions (all useCallback):**

1. `loadAssessment()` — called from useEffect on mount, returns saved profile or null
2. `setSelfRating(skill, level, confidence)` — updates selfRatings for a skill
3. `getQuizQuestions(skill)` — returns the 3 hardcoded questions for the given skill
4. `submitQuizResult(skill, answers)` — takes `{q1: 'A', q2: 'B', q3: 'C'}`, scores against correct answers, updates quizResults
5. `computeFinalLevel(skill)` — after quiz complete:
   - If quiz score = 3/3 AND self-rating matches → final = self-rating, method='self'
   - If quiz score = 3/3 AND self-rating differs → average with rounding, method='adjusted'  
   - If quiz score = 2/3 → use self-rating, method='adjusted' (slight downward bias if self was higher)
   - If quiz score ≤ 1/3 → use quiz equivalent (map score to level: 0→1, 1→2), method='quiz'
6. `completeAssessment()` — computes all finalLevels, sets completedAt, persists to IndexedDB
7. `resetAssessment()` — clears assessment data to start fresh
8. `getOverallDse()` — derives overall from finalLevels using computeOverallDseLevel pattern from dseGrading
9. `hasCompletedAssessment` — boolean derived from completedAt being non-null

**Persistence:**
- Load from DSE_KEYS.ASSESSMENT in useEffect on mount (single load via loadedRef flag, same pattern as useSkillAnalytics)
- `persist(profile)` is called after each mutation (setSelfRating, submitQuizResult, completeAssessment) — debounce NOT needed since assessment is not high-frequency
- Use try/catch with silent failure for IndexedDB operations (existing pattern)

**Exports:**
```js
return {
  profile, isLoaded, activeSkill, setActiveSkill,
  currentPhase, setCurrentPhase,
  setSelfRating, getQuizQuestions, submitQuizResult,
  computeFinalLevel, completeAssessment, resetAssessment,
  hasCompletedAssessment, getOverallDse,
  // Derived
  selfRatings: profile.selfRatings,
  quizResults: profile.quizResults,
  finalLevels: profile.finalLevels,
};
```

**Code style:**
- All functions use `useCallback` with correct deps
- useRef for mount guard (loadedRef)
- useState for profile, isLoaded, activeSkill, currentPhase
- No useMemo needed (small data, infrequent reads)
- Default export, named function
- Import useIndexedDB from './useIndexedDB'
- Import { scoreToDseLevel, computeOverallDseLevel } from '../utils/dseGrading'
